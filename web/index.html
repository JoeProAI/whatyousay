Quickstart

Install, pick your two languages, and start talking. First launch pulls the model pack matched to your device. After that, the network is never touched again.

# the app ships with a stub pipeline, then pulls real models on first run
whatyousay pull hy-mt2 whisper-small piper
whatyousay run --offline # prove it: no network, full pipeline

Models are verified by hash before they are trusted, and stored in app-private space. You can swap the translation engine without touching the app.
